# 🔐 Password Manager

Менеджер паролей с генератором для безопасного хранения и управления паролями.

## 📋 Описание

Веб-приложение для генерации сложных паролей и их сохранения. Помогает создавать надёжные пароли и хранить их в одном месте.

## ✨ Функциональность

- Генерация паролей с настройками (длина, цифры, спецсимволы)
- Сохранение паролей с привязкой к сайту и логину
- Просмотр списка всех паролей
- Поиск по сайту или логину
- Просмотр и копирование паролей
- Удаление паролей
